Font Setting and 4k Support

Font preference

User can adjust font size, family and color for different application UI, there are 5 different UI font settings, if user has a high-resolution or 4k monitor and feel font are too small, they can adjust the font size accordingly using the following settings.

  1. Application
  2. CLI
  3. Workspace
  4. Activity Wizard
  5. Button/Labels

 

Application Font

The Application Font changes the size of font and icons for the main Application UI and dialogs. Use the slider to adjust the font and icon size then click Apply.

 

 

CLI Font

The CLI font setting changes font size and family in the terminal.

 

 

Workspace Font

The Workspace font setting changes the font size and family in the Logical and Physical Workspace.

 

 

Activity Wizard Font

The Activity Wizard font setting changes the font size and family in the Activity Wizard.

 

 

Button/Labels Font

The Button/Labels font setting changes the font size and family in Desktop and Server dialogs.
